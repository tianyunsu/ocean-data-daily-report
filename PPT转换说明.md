## 如何转换为PPT格式

### 方法1：使用浏览器打印功能（推荐）

1. **打开HTML文件**
   - 文件已在浏览器中打开

2. **打印为PDF**
   - 按 `Ctrl + P` 打开打印对话框
   - 目标打印机选择 "另存为PDF" 或 "Microsoft Print to PDF"
   - 布局选择 "纵向"
   - 勾选 "背景图形"
   - 点击保存

3. **转换为PPT**
   - 使用在线工具（如 Smallpdf、ILovePDF 等）
   - 或使用 Adobe Acrobat Pro 将PDF导出为PPT
   - 或在 PowerPoint 中直接打开PDF

### 方法2：使用PowerPoint导入

1. **打开PowerPoint**
   - 创建新的空白演示文稿

2. **使用"设计灵感"功能**
   - 点击"设计" → "设计灵感"
   - 可能会自动识别并转换为幻灯片

3. **手动复制内容**
   - 从HTML复制内容到PowerPoint
   - 保持相同的配色方案

### 方法3：使用在线HTML转PPT工具

1. 访问以下在线工具：
   - https://www.zamzar.com/convert/html-to-ppt/
   - https://convertio.co/zh/html-ppt/
   - https://cloudconvert.com/html-to-ppt

2. 上传 `presentation.html` 文件

3. 选择输出格式为 PPTX

4. 下载转换后的文件

### 方法4：使用Chrome扩展

1. 在Chrome浏览器中安装扩展：
   - "Full Page Screen Capture"
   - "GoFullPage"

2. 截取每一页幻灯片

3. 将截图插入PowerPoint

## 配色方案参考

如果你需要在PowerPoint中手动创建，可以使用以下配色：

| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 中国红 | #C8102E | 主色、标题、边框 |
| 金色 | #FFD700 | 装饰线、强调 |
| 深红色 | #8B0000 | 小标题、强调文字 |
| 浅米色 | #FFF5E6 | 背景色 |
| 白色 | #FFFFFF | 卡片背景 |
| 深灰色 | #333333 | 正文文字 |

## 字体建议

- 标题：黑体 (SimHei) 或 微软雅黑加粗
- 正文：微软雅黑 (Microsoft YaHei)
- 数字：Arial

## 注意事项

1. **保持简洁**：不要添加过多的动画和过渡效果
2. **统一风格**：所有幻灯片使用相同的配色和字体
3. **内容优先**：确保文字清晰可读，字号不小于18pt
4. **留白适当**：不要填满每一寸空间
