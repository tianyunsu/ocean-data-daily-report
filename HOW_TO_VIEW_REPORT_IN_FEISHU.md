## 🔧 故障排查

### 问题 1：找不到日报消息

**解决方案：**
1. 检查是否在正确的群组中
2. 向上滚动查看历史消息
3. 检查消息通知设置（确保未被静音）
4. 尝试刷新群组

### 问题 2：无法打开飞书文档链接

**解决方案：**
1. 确认网络连接正常
2. 确认已登录飞书账户
3. 确认对文档有访问权限
4. 尝试复制链接在浏览器中打开

### 问题 3：HTML 报告无法打开

**解决方案：**
1. 确认文件路径正确
2. 尝试用不同的浏览器打开
3. 检查浏览器是否支持 HTML5
4. 尝试用文本编辑器查看源代码
